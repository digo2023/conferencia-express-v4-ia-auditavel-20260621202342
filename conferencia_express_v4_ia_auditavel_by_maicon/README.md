# Conferência Express V4 — IA Auditável By Maicon

Aplicativo Android nativo para conferência operacional da Express-Unidade Colorado.

## Principais recursos

- Conferência Analítico x Ordem de Compra, item por item.
- Módulos: proteínas, perecíveis, hortifruti, químicos e conferência total.
- Modo IA: Rápida, Segura e Auditoria.
- Tela de revisão manual obrigatória antes do relatório final.
- Vincular manualmente item do analítico com item encontrado na O.C.
- Salvar equivalências aprendidas para próximas conferências.
- Corrigir quantidade entregue manualmente.
- Remover item do relatório final.
- Ver motivo detalhado da classificação.
- Conversão de embalagens da O.C. para unidade base.
- Banco local SQLite de histórico, pendências, notificações, aprendizado e equivalências.
- Alertas preventivos às 06:00 e 12:00 com pendências vencidas, 24h, 48h e 72h.
- Relatório PDF revisado, Excel e resumo WhatsApp.
- Bloqueios de segurança para datas fora do período e extração baixa.

## Segurança operacional

A IA sugere, prioriza e aprende, mas o relatório final só é gerado após confirmação manual. Itens duvidosos devem ser vinculados ou corrigidos antes de emitir o PDF.

## Build

O APK é gerado pelo GitHub Actions no workflow `Build APK Android Nativo - Conferência Express`.
